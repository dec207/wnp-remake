# The Wizard and the Princess (WNP) - Python Remake

1980년대 고전 어드벤처 게임 **'The Wizard and the Princess'**를 현대적인 Python 텍스트 어드벤처 엔진으로 재구현하는 프로젝트입니다.

## 🎮 게임 개요
사악한 마법사에게 납치된 공주를 구하기 위해, 마법사의 동굴과 사막, 성을 모험하며 퍼즐을 풀고 적을 물리쳐야 합니다.

- **장르**: 텍스트 어드벤처 (MUD 스타일)
- **언어**: Python 3.x
- **특징**:
  - 자연어 명령어 파싱 (`GO NORTH`, `THROW POD`)
  - 실시간 상태 관리 (점수, 골드, 식량)
  - 확장 가능한 데이터 주도(Data-Driven) 아키텍처

## 🚀 실행 방법

### 요구 사항
- Python 3.8 이상

### 설치 및 실행
```bash
# 1. 저장소 클론
git clone https://github.com/your-repo/wnp-remake.git
cd wnp-remake

# 2. 게임 실행
python3 -m src.main
```

## 🗺️ 프로젝트 구조
- `src/`: 게임 소스 코드
- `docs/`: 설계 및 기획 문서
- `tests/`: 테스트 코드
